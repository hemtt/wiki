# Arma 3 Wiki

A project to store data about Arma 3 commands in a useable format.

The [dist](https://github.com/hemtt/wiki/tree/dist) branch contains all the parsed files.

Versions 0.4.x and below used the [dist](https://github.com/hemtt/wiki/tree/dist) branch.
